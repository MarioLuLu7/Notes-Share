# Math 方法

1. `Math.random() // '一个随机数'`

   生成一个 0~1 的一个随机数

2. `Math.ceil(2.3) // 3`

   向上取整

3. `Math.floor(2.3) // 2`

   向下取整

4. `Math.PI // 3.141592653589793`

   π

5. `Math.max(1,23,45) // 45`

   最大值

6. `Math.min(1,23,45) // 1`

   最小值

7. `Math.E // 2.718281828459045`

   e

8. `Math.round(231.1231231) // 231`

   去掉小数位

9. `Math.abs(-2) // 2`

   绝对值

10. `Math.sqrt(4) // 2`

开平方

11. `Math.cbrt(8) // 2`

开立方

12. `Math.pow(2,4) // 16`

x 的 y 次方
